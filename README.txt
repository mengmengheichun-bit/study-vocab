Cách dùng trên GitHub Pages:

1. Upload 2 file này lên cùng một chỗ trong repo:
   - index.html
   - vocab.json

2. Bật GitHub Pages.

3. Khi muốn thêm/sửa từ để mọi người đều thấy:
   - Sửa file vocab.json trên GitHub.
   - Hoặc vào web, thêm/import từ, bấm Export JSON, rồi upload file vocab.json mới lên GitHub thay file cũ.

Lưu ý: dữ liệu thêm trực tiếp trên web chỉ lưu ở trình duyệt của người đang dùng. Muốn ai mở link cũng thấy thì phải cập nhật vocab.json trên GitHub.
